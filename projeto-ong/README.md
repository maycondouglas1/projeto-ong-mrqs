# Projeto ONG - Plataforma Web

Projeto desenvolvido para a disciplina de Desenvolvimento Web, criando uma plataforma completa para ONGs com HTML5 semântico.

## Estrutura do Projeto

```
projeto-ong/
├── index.html          # Página inicial com informações institucionais
├── projetos.html       # Página de projetos, voluntariado e doações
├── cadastro.html       # Formulário de cadastro completo
└── assets/
    └── images/         # Pasta para imagens do projeto
```

## Páginas

### index.html
- Informações sobre a organização
- Missão, visão e valores
- Histórico e conquistas
- Equipe e estrutura organizacional
- Dados de contato

### projetos.html
- Projetos sociais em andamento
- Oportunidades de voluntariado
- Como fazer doações
- Metas e progresso das campanhas

### cadastro.html
- Formulário completo de cadastro
- Validação nativa HTML5
- Campos obrigatórios marcados
- Máscaras de input para CPF, telefone e CEP

## Recursos Implementados

- ✅ HTML5 semântico completo
- ✅ Estrutura com header, nav, main, section, article, footer
- ✅ Formulário com validação nativa
- ✅ Máscaras de input (CPF, telefone, CEP)
- ✅ Meta tags para SEO
- ✅ Hierarquia de títulos lógica
- ✅ Imagens em todas as páginas
- ✅ Acessibilidade básica (aria-required, labels adequados)

## Validação

Os arquivos HTML foram desenvolvidos seguindo os padrões W3C e podem ser validados no [W3C Validator](https://validator.w3.org/).

## Observações

- Este projeto utiliza apenas HTML5 puro, sem CSS ou JavaScript
- As imagens referenciadas precisam ser adicionadas na pasta `assets/images/`
- O formulário utiliza validação nativa do HTML5
